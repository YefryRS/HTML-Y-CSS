ALPHA Y OMEGA

Alfa es la primera letra del alfabeto griego y omega es la última. El griego se usó para escribir la parte de la Biblia que conocemos como Nuevo Testamento, que incluye el libro de Revelación o Apocalipsis. La posición de estas dos letras en el alfabeto griego se usa para indicar que solo Jehová es el principio y el fin.
